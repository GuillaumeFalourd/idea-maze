## Contribute

Contributions are welcome. Thanks!

Install `depot_tools`:
```shell
cd $HOME && git clone https://chromium.googlesource.com/chromium/tools/depot_tools.git
```

Add `depot_tools` to the end of your PATH:

~/.bashrc
```patch
+export PATH="$PATH:${HOME}/depot_tools"
```
